## Version 0.4.0 Release



## HTTPS Support Added

**Key Features**

**1. Easy Configuration**

```javascript
javascript// HTTP (default)
await build({ cache: { type: 'memory' } });

// HTTPS (just add protocol + ssl)
await build({
  protocol: 'https',
  ssl: {
    key: fs.readFileSync('./key.pem'),
    cert: fs.readFileSync('./cert.pem')
  },
  cache: { type: 'memory' }
});
```

**2. Complete Feature Parity**<br>
All features work identically:

✅ Routing<br>
✅ Middleware<br>
✅ Caching (all databases)<br>
✅ Throttling<br>
✅ Logging<br>
✅ Error tracking<br>
✅ Cookies<br>
✅ All extensions<br>

**3. Dual-Mode Support**
```javascript
// Run both simultaneously
const http = await build({ protocol: 'http' });
http.listen(80);

const https = await build({
  protocol: 'https',
  ssl: { key, cert }
});
https.listen(443);
```

**4. Helpful Errors**
```javascript
javascriptawait build({ protocol: 'https' });
// Error: HTTPS requires ssl.key and ssl.cert in options.
// Example: build({ protocol: "https", ssl: { ... } })
```
